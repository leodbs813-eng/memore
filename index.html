메 (ME) 방식

정답을 머릿속으로 떠올려 스스로 평가하거나 물리 정렬하는 자가 진단형 학습

  • 암기: 앞면을 보고 답을 떠올린 후, 카드를 뒤집어 확인합니다.
  • 배열: 항목 카드를 손으로 드래그하여 올바른 순서로 재배치합니다.
  • 분류: 제시 카드가 하단 어떤 대분류 그룹에 속하는지 매칭합니다.
모 (MO) 방식

답안과 세부 정의를 직접 작성하여 점검하는 완성형 검증 학습

  • 암기: 인풋 폼에 카드 뒷면의 정답을 완벽히 타이핑해야 정답 인정됩니다.
  • 배열: 순서 배치 후, 각 항목의 뜻과 상세 쓰임새를 직접 입력해야 합니다.
  • 분류: 일반 분류와 동일하게 진행되며 터치 힌트를 활용해 카드를 구분할 수 있습니다.
RE 방식 (거꾸로 학습)

기존의 앞/뒤, 명칭/뜻 구조를 완벽하게 뒤바꿔 출력해 뇌에 새로운 역방향 인지 자극을 주는 심화 학습

  • 암기: 뜻(뒷면)이 앞면에 먼저 출력되며, 사용자는 원래 단어(앞면)를 직접 타이핑하여 검증합니다.
  • 배열: 카드가 항목의 뜻(설명)으로 먼저 노출되며, 사용자는 그에 알맞은 원래 '명칭'을 입력해야 정답 처리됩니다.
  • 분류: 카드 앞면에 원래 상세 설명(뜻)이 나타나며, 카드를 뒤집어야 원래 카드명(이름)을 볼 수 있습니다.
