메 (ME) 방식

정답을 머릿속으로 떠올려 스스로 평가하거나 물리 정렬하는 자가 진단형 학습

  • 암기: 앞면을 보고 답을 떠올린 후, 카드를 뒤집어 확인합니다.
  • 배열: 항목 카드를 물리적으로 손으로 드래그하여 올바른 순서로 재배치합니다.
  • 분류: 화면의 제시 카드가 하단 어떤 분류 그룹에 속하는지 선택지로 매칭합니다.
모 (MO) 방식

답안과 세부 정의를 직접 작성하여 점검하는 완성형 검증 학습

  • 암기: 인풋 폼에 카드 뒷면의 정답을 완벽히 타이핑해야 정답 인정됩니다.
  • 배열: 순서대로 항목을 드래그 배치한 후, 각 항목의 뜻과 상세 쓰임새까지 직접 서술해내야 인정됩니다.
  • 분류: 카드 분류 매칭을 위해 카드를 터치하여 뒷면 힌트를 보며 매칭시킵니다.
